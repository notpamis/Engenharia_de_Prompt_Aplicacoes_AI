explique o que é uma API

Uma **API** (do inglês *Application Programming Interface*, ou **Interface de Programação de Aplicações**) é um conjunto de regras que permite que **dois sistemas diferentes se comuniquem**.

### Explicando de forma simples:

Uma API funciona como um **garçom em um restaurante**:

* Você (usuário ou programa) faz um pedido.
* O garçom (API) leva o pedido para a cozinha (servidor/sistema).
* A cozinha prepara a resposta.
* O garçom traz a resposta para você.

### Exemplo real

Quando você usa um aplicativo de clima no celular, ele não “adivinha” a previsão do tempo. Ele faz uma requisição para uma API de clima, que devolve informações como:

* temperatura
* chance de chuva
* vento
* umidade

### Para que serve uma API?

Ela serve para:

* buscar dados (ex: clima, preços, mapas)
* enviar dados (ex: cadastro, login)
* integrar sistemas (ex: site conectando com banco, pagamento, WhatsApp)

### Exemplo comum: API de login

Um aplicativo pode mandar para a API:

* email: `teste@gmail.com`
* senha: `12345`

E a API responde:

* "usuário autenticado"
* ou "senha incorreta"

Resumindo: **API é o meio que permite que programas conversem entre si de forma organizada e segura.**
